1) CyberShield - Android Cybersecurity Assistant
CyberShield is an Android application designed to improve mobile security by detecting scams, analyzing risk levels, checking suspicious URLs, and educating users about cyber threats through an interactive quiz.

2) Features

->  Scam SMS Detection
->  Malicious URL Checker
-> AI-based Risk Assessment using TensorFlow Lite
-> Permission Scanner
-> Security Dashboard
-> SMS Log Analysis
-> Cybersecurity Quiz
-> High Risk Activity Detection


3) Tech Stack
1.Kotlin
2.Android Studio
3.SQLite
4.TensorFlow Lite
5.XML
6.Gradle


4) Project Structure

app/
 ├── activities
 ├── scanner
 ├── risk
 ├── assets
 ├── layouts
 └── resources


5) Installation

1. Clone the repository:
git clone https://github.com/shlokasharma/CyberShield-Android-App.git
2. Open in Android Studio
3. Sync Gradle
4. Run on an emulator or Android device.
